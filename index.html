AgWe

Open source, hardware agnostic grow automation for all!

Download this project as a .zip file Download this project as a tar.gz file

The goal of the AgWe project is to drastically lower the costs of automation in climate controlled agriculture. Open source software has revolutionized many industries, but food remains stuck in the past. Overpriced proprietary hardware and software continues to dominate the industry. It just doesn't make sense. Spend less and grow more with AgWe.

The first stage of the AgWe project is simple. Baby steps, right?

1.Create lightweight software to run on readily available microcontroller platforms that will expose climate data via Websockets. Target controllers include:

  • Tessel 2 via agwe-tessel
  • Raspberry PI
  • Arduino
  • Espruino
  • Intel Edison

...there are many more...we'd love contributions

2.Build a cross platform controller software that can discover, connect to the microcontrollers, store, and visualize the data.
